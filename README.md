# 🌌 Void Origin | Intelligence Core

A minimalist, high-performance AI chat interface built with **Ollama (Llama 3.1)**. On the surface, it appears to be a standard, professional "Intelligence Core," but it contains a hidden **Void Protocol** that overwrites reality through specific command triggers.

## 🚀 Key Features

* **Dual-State UI:** Toggle between a clean, Apple-inspired **Stable Mode** and a dark, glowing **Void Mode**.
* **Reality-Shatter Transition:** Custom CSS animations including screen shakes, "void cracks," and lighting shifts.
* **Command Interceptor:** A custom JavaScript logic gate that catches system commands before they reach the AI.
* **Cinematic Audio:** Integrated sound effects for the "Shatter" and "Vent" sequences.
* **Dynamic Tab Control:** The browser tab title and favicon transform automatically when the Void is manifest.
* **Local AI Power:** Fully integrated with the Ollama API for private, local LLM responses.

## 🛠️ Technical Stack

* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript (ES6+).
* **AI Engine:** Ollama running `llama3.1`.
* **Animations:** CSS Keyframes & Cubic-bezier transitions.
* **Styling:** Custom property (CSS variables) architecture for instant theme swapping.

## 🗝️ System Commands

| Command | Action | Visual Effect |
| :--- | :--- | :--- |
| `he who remains is null` | **Activate Void** | Screen shatter, Purple glow, Tab rename, Audio fire. |
| `time to cooldown` | **Deactivate Void** | Cyan heat vent, Reset to white, Tab restore, Audio fire. |

## 📂 Project Structure

```text
├── index.html       # Structural layout & UI components
├── style.css        # Master stylesheet (Stable & Void themes)
├── script.js        # Core logic, AI integration, & Audio triggers
├── shatter.mp3      # Activation sound effect
└── vent.mp3         # Deactivation sound effect
```

## 📜 How to Run

1.  Ensure **Ollama** is installed and running locally.
2.  Pull the model: `ollama run llama3.1`.
3.  Open `index.html` in any modern web browser.
4.  Interact with the Core. If necessary, use the "Invasion" command to reveal the true interface.
